# CineHub

Modern movie discovery platform built with HTML, CSS, and vanilla JavaScript.

## Features

- Browse popular & trending movies
- Real-time search
- Genre / year / rating filters
- Responsive movie grid & list views
- Detailed movie modal with cast
- YouTube & IMDb trailer support
- Watchlist (localStorage)

## Tech Stack

- TMDB API
- YouTube Data API v3 (fallback)
- Font Awesome 6
- Google Fonts (Inter)
